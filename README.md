# Plumb Line Observatory

Connecting the patterns behind the narratives.

An interactive 3D observatory for analyzing manipulation patterns and biblical principles in modern discourse.

## Tech Stack

- React
- Vite
- Three.js

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Pixeloute/plumbline.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Deployment

Connected to Vercel for automatic deployment on push to `main`.
