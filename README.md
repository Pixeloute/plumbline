# The Plumb Line Observatory

A critical epistemology instrument for mapping manipulation patterns and biblical principles in modern discourse.

## Phase 2: Multi-Lens Viewing

The Observatory now supports four distinct analytical views to surface structural truths across ideological boundaries:

- **3D Sphere (1)**: The visual flagship for spatial orientation.
- **2D Force Graph (2)**: Reveals clustering and connection density.
- **Radar Fingerprint (3)**: Structural comparison of node profiles.
- **Timeline Scrubber (4)**: Historical context and discourse frequency.

## Keyboard Shortcuts

- `1`-`4`: Switch View Modes
- `C`: Toggle Tribe Clustering (Force Graph)
- `L`: Toggle Node Labels (Force Graph)
- `Esc`: Clear Selection / Reset View

## Tech Stack
- React 19 + Three.js + D3 + Vite
- Code-splitting with Lazy Loading for visual modules

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Deployment

Connected to Vercel for automatic deployment on push to `main`.
